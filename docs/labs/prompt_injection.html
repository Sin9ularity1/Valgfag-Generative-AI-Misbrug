Prompt Injection Lab


What is Prompt Injection

Prompt Injection is the act of giving an LLM a malicious payload, either as an act of malice or accidentally, that alters the way the LLM behaves.

Prompt Injection could cause LLM's to violate guidelines, disclose sensitive information, generate harmful content, enable unauthorized access or influence critical decisions.

Types of Prompt Injection Vulnerabilities

Direct Prompt Injections
The model's behaviour is altered by the users direct set of inputs
Indirect Prompt Injections
The LLM accepts input from external sources like websites or files that alter the models behaviour.
